Official Bangladesh Ramadan Schedule with CSV import and real-time countdown.

== Description ==
Accurate Sehri and Iftar timings for Bangladesh based on Islamic Foundation. Supports all districts and real-time countdown.

== Installation ==
1. Upload the `ramadan-timer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode `[ramadan_timer]` for the full widget or `[ramadan_countdown]` for the lite version.
